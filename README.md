## 설치 및 실행 방법
1. 저장소 클론
   ```
   git clone https://github.com/username/repository.git
   ```
2. 프로젝트 폴더로 이동
   ```
   cd repository
   ```
3. 의존성 설치
   ```
   npm install
   ```
4. 환경 변수 설정
   ```
   cp .env.example .env
   ```
   그리고 `.env` 파일에 필요한 환경 변수를 설정합니다.
5. 개발 서버 실행
   ```
   npm start
   ```
